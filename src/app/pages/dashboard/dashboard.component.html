About

I’m an SQA Engineer with 3+ years of experience testing web applications, currently working with Efani, a product-based company based in California, USA. I joined as their first QA hire and built their entire QA process from the ground up — including test cases, regression suites, and bug tracking workflows.

Since Efani’s platform is built in Angular, I worked closely with TypeScript and the frontend codebase too, which is how I picked up frontend development alongside QA. I also automated their website testing using Playwright and built out the CI/CD pipeline for the repository, significantly reducing manual regression time.

Before Efani, I worked at K2X Technologies and Workbox, handling manual functional, integration, and regression testing across enterprise platforms. My core toolkit includes API testing with Postman, SQL, Jira, and CI/CD validation — with Angular and Playwright rounding out the frontend and automation side.
